1 Resumo

O curso ensina a usar Inteligência Artificial e Machine Learning na análise de dados econômicos e financeiros com Python. O ponto de partida são os fundamentos de engenharia de prompt e o uso de ferramentas como Google AI Studio e Gemini. A partir daí, o conteúdo avança para aplicações em otimização de portfólios, previsões econômicas, análise de séries temporais e análise de sentimento.

A abordagem é prática. Cada tema é trabalhado com código em Python sobre problemas concretos, como prever retornos de ações, identificar crises financeiras, projetar o consumo de energia ou avaliar o tom da política monetária. O objetivo é que o aluno saia capaz de desenvolver, treinar e aplicar modelos de IA e Machine Learning no próprio trabalho.

O curso é voltado a economistas, analistas financeiros e cientistas de dados que já programam em Python e querem incorporar IA e Machine Learning ao seu fluxo de análise.

2 Pré-requisitos e Objetivos

Pré-requisitos

É necessário conhecimento prévio em programação Python para acompanhar o conteúdo. As aulas usam Python como ferramenta principal de implementação, com apoio do Google Colab.

O que você será capaz de fazer

  • Aplicar técnicas de engenharia de prompt para obter resultados úteis de modelos de IA.
  • Usar ferramentas como Google AI Studio e Gemini em tarefas de análise de dados.
  • Desenvolver, treinar e implementar modelos de IA e Machine Learning em Python.
  • Aplicar Machine Learning a problemas econômicos e financeiros, como previsão de retornos, otimização de portfólios e projeção de séries temporais.
  • Analisar texto com IA, incluindo relatórios de gestores e o tom da política monetária.
  • Automatizar a geração de relatórios com apoio de IA.

Habilidades e ferramentas

  • Linguagem: Python
  • Ambiente: Google Colab
  • Ferramentas de IA: Google AI Studio, Google Gemini, TimeGPT
  • Áreas de aplicação: engenharia de prompt, otimização de portfólios, previsão econômica, análise de séries temporais, análise de sentimento e geração automática de relatórios
