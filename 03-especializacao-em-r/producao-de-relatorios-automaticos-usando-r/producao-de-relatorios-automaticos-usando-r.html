Produção de Relatórios Automáticos usando R

Programa e informações gerais do curso

Autor

Análise Macro

1 Resumo

Este curso ensina a criar relatórios e apresentações do zero com a linguagem R. A proposta é prática: você percorre todas as etapas do ciclo de análise de dados, da extração dos dados ao documento final compilado, usando o Quarto como ferramenta central de publicação.

O conteúdo é voltado para quem já programa em R e quer centralizar a análise e a comunicação dos resultados em um único fluxo de trabalho. Em vez de montar relatórios manualmente a cada atualização, você aprende a construir documentos reproduzíveis que podem ser gerados de novo sempre que os dados mudam.

A última parte do curso trata de automatização. Com Git, GitHub e GitHub Actions, você configura um fluxo que executa o código, renderiza o relatório e atualiza o repositório sem intervenção manual, em um horário definido ou a cada alteração do projeto.

2 Pré-requisitos e Objetivos

Pré-requisitos

Para o melhor aproveitamento, é esperado que o interessado cumpra os seguintes requisitos:

  • Computador com acesso à internet;
  • Cadastro de conta gratuita no Posit Cloud;
  • Cadastro de conta gratuita no GitHub;
  • Nível introdutório/intermediário de R;
  • Noções básicas de controle de versão Git;
  • Domínio de inglês para leitura.

O que você será capaz de fazer

  • Construir relatórios reproduzíveis e automáticos com a linguagem R;
  • Identificar as ferramentas adequadas para produção de relatórios e apresentações;
  • Produzir relatórios e apresentações do zero, da extração dos dados à compilação;
  • Automatizar a geração e o envio de relatórios e apresentações.

Habilidades e ferramentas

  • R: extração de dados, computação e geração dos resultados;
  • Quarto: produção de relatórios, apresentações e outros formatos de publicação;
  • revealjs: formato de apresentações dentro do Quarto;
  • Posit Cloud: ambiente de programação na nuvem;
  • Git e GitHub: controle de versão e repositório do projeto;
  • GitHub Actions: automatização do fluxo de execução e renderização.
