1 Resumo

O curso Python para Investimentos ensina a linguagem Python com foco no mercado financeiro, partindo do zero. Começa pelos fundamentos da linguagem e avança até projetos aplicados, em que cada problema do dia a dia de investimentos é resolvido com código.

A estrutura segue dois blocos. No primeiro, você aprende a programar em Python e a usar as principais bibliotecas de análise de dados. No segundo, aplica esse conhecimento em projetos completos, que percorrem todo o ciclo de trabalho: coleta e tratamento de dados, análise, modelagem e comunicação dos resultados.

É indicado para quem deseja entrar no mercado financeiro e para profissionais que querem atualizar seus conhecimentos com Python. Ao final, você terá scripts prontos para tarefas como construir carteiras, estimar betas, medir volatilidade, montar a estrutura a termo de juros e integrar análises ao MetaTrader.

2 Pré-requisitos e Objetivos

Pré-requisitos

Não é necessário conhecimento prévio de programação, pois o curso parte do zero. Para quem quiser aprofundar a base antes ou em paralelo, a Análise Macro recomenda os cursos Programação em Python para Análise de Dados, Teoria Macroeconômica usando Python, Análise de Conjuntura usando Python e Estatística para Análise de Dados usando Python.

O que você será capaz de fazer

  • Programar em Python do zero, com domínio das estruturas de dados e do controle de fluxo.
  • Importar dados de APIs, planilhas Excel, PDFs e arquivos proprietários.
  • Visualizar dados e comunicar resultados de análises.
  • Aplicar modelos estatísticos e de machine learning a problemas de investimento.
  • Construir carteiras de investimentos, estimar o beta de mercado e medir volatilidade.
  • Montar a estrutura a termo da taxa de juros e calcular juro real e juro neutro.
  • Desenvolver estratégias como pair trading, médias móveis e indicadores fundamentalistas.
  • Integrar análises feitas em Python com a plataforma MetaTrader.

Habilidades e ferramentas

  • Linguagem: Python.
  • Ambiente: Google Colab.
  • Manipulação de dados: estruturas de dados (vetores, listas, matrizes e dataframes) e importação a partir de APIs, Excel, PDFs e arquivos proprietários.
  • Visualização: seaborn e matplotlib.
  • Modelagem: regressão linear, ARIMA, dados em painel e scikit-learn.
  • Finanças: pyfolio para análise de portfólio e integração com o MetaTrader.
